# Simon Preece Portfolio

This file is included in the GitHub Pages deployment branch so the deployed branch contains a README for the generated site.

## Site details

- Built with React, TypeScript, and Vite
- Includes light/dark theme switching
- Deployed to GitHub Pages using `gh-pages`

## Deployment

The `gh-pages` branch is generated from the `dist` folder when running:

```bash
npm run deploy
```

## Notes

This file is copied into the `dist` output as part of the Vite build, so it is available in the deployed branch.
